Наши вопросы

Нужно ли платить за доступ к сервису, если у компании не планируются командировки в ближайший месяц? +

Нет — вы можете подключить демо-доступ и изучить сервис без обязательств. Оплата начинается только при переходе на рабочий тариф, когда поездки снова появятся в планах.

Есть ли преимущества в приобретении билетов через Smartway? +

Да: единый кабинет, контроль по тревел-политике, прозрачная отчётность, поддержка 24/7 и удобные схемы оплаты для компании.

Можно ли использовать несколько тревел-политик одновременно? +

Да. Политики можно настроить для разных подразделений, ролей или категорий сотрудников и комбинировать правила в рамках одной организации.

Как работает интеграция с 1С? +

Smartway обменивается данными с 1С: заказы, документы и справочники синхронизируются по готовым сценариям, без ручного дублирования в учётной системе.

В нашей группе компаний несколько юридических лиц. Нужен ли отдельный личный кабинет для каждой компании? +

Обычно достаточно одной структуры доступа с разделением по юрлицам внутри сервиса — уточните с менеджером модель, которая подойдёт вашей группе.
